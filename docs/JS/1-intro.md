# JavaScript 入门指南

## JavaScript 简介

JavaScript 是一种广泛应用于 Web 开发的脚本语言，由 Brendan Eich 在 Netscape 公司开发。它是一种动态类型、基于原型的面向对象编程语言，能够为网页添加交互性和动态内容。JavaScript 作为 Web 的三大核心技术之一（HTML、CSS 和 JavaScript），已成为现代 Web 应用程序开发的基础。
